InfraDesk provides a centralized digital platform where campus issues can be reported, tracked, and managed in real time. The system introduces structured reporting, role-based access for administrators, AI-assisted recommendations, and geographic visualization of issues across the campus.
The platform is live and deployed using Firebase Hosting.

Key Features :
- Issue reporting with category, location, and description  
- Real-time display of reported issues  
- Admin-controlled status updates (Reported / In Progress / Resolved)  
- Automatic timestamps for issue reporting and status changes  
- AI-based action recommendations based on issue category  
- Search and category-based filtering of issues  
- Interactive campus map showing issue locations using Google Maps  
- Secure and scalable backend using Firebase Firestore  


Tech Stack :
Frontend :  HTML5, CSS3, JavaScript (ES Modules)
Backend & Database: Firebase Firestore (NoSQL database), Firebase Hosting
Google Technologies: Google Maps JavaScript API, Google Geocoding API, Firebase SDK

AI Integration: Rule-based AI recommendation system (Gemini-ready for future enhancement)
